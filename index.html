OpenCode Systemd

A systemd service manager for OpenCode with automatic daily upgrades. One-line install, zero configuration needed.

Install

curl -fsSL https://ocsd.nbr.st/install.sh | bash

[*] Features

One-Line Install

Get OpenCode running as a systemd service in seconds with automatic configuration.

Auto-Upgrades

Daily automatic updates via systemd timers. Configurable schedule, default 5 AM.

Web UI Ready

Runs OpenCode web interface on localhost with auto-restart on failure.

CLI Wizard

Interactive management tool for status, logs, upgrades, and configuration.

[*] Commands

opencode-systemd status Check service status and configuration
opencode-systemd upgrade Run manual upgrade and restart services
opencode-systemd logs View service logs in follow mode
opencode-systemd update Update configuration or restart services
opencode-systemd uninstall Remove all systemd services
