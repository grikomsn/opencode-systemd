OpenCode Systemd

Systemd service manager for OpenCode with automatic daily upgrades

🚀 One-Line Install

Get OpenCode running as a systemd service in seconds

🔄 Auto-Upgrades

Daily automatic updates via systemd timers

⚙️ Configurable

Customize host, port, and upgrade schedule

🔧 CLI Wizard

Interactive management tool included

Quick Install

Run this command in your terminal:

curl -fsSL https://ocsd.nbr.st/install.sh | bash

After installation, access OpenCode at http://127.0.0.1:4096

CLI Commands

# Check status
opencode-systemd status

# Manual upgrade
opencode-systemd upgrade

# View logs
opencode-systemd logs

# Show all commands
opencode-systemd --help
